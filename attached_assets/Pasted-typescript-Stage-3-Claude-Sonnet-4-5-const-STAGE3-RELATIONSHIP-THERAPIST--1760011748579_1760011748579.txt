### 1. 패턴의 심리적 뿌리

FBI가 발견한 각 패턴(P1, P2, P3...)마다 분석:

**분석 질문:**
- 왜 이런 패턴이 생겼나?
- 무의식적 동기는?
- 어떤 심리적 욕구를 채우려는가?
- 어떤 두려움을 피하려는가?

**해석 구조:**
FBI 발견: "감정 억압 후 폭발" 패턴 (5회)
심리치료사 해석:
→ 근본 원인: 갈등 회피형 소통
→ 무의식적 동기: 거절/버림받음 두려움
→ 단기 이득: 즉각적 평화 유지

→ 핵심 욕구: 안전한 애착

### 2. 관계 역학 분석

**시스템 관점:**
- 관계는 **상호작용 시스템**
- 한 사람 행동 → 상대방 반응 → 순환
- Circular Causality (순환 인과)

**발견해야 할 역학:**

**A. 추격-도피 (Pursuer-Distancer)**
- 증거: PURSUIT 태그 vs AVOIDANCE 태그
- 한쪽 다가가면 → 다른쪽 물러남
- 불안 증가 → 더 추격 → 더 도피 → 악순환

**B. 비난-방어 (Criticize-Defend)**
- 증거: CONFLICT 태그 패턴
- 한쪽 비난 → 다른쪽 방어
- 서로 피해자 느낌 → 소통 단절

**C. 과잉-저기능 (Overfunctioning-Underfunctioning)**
- 증거: SUPPORT 태그 불균형
- 한쪽 모든 책임 → 다른쪽 의존
- 불균형 심화 → 피로/무력감

**분석 구조:**
발견된 역학: 추격-도피
트리거: {userName} FEAR 태그 → PURSUIT
반응: {partnerName} AVOIDANCE 태그
결과: {userName} 불안 증가 → 더 강한 PURSUIT
악순환: 거리 더 멀어짐
심리적 배경:

{userName}: 불안 애착 가능성
{partnerName}: 회피 애착 가능성

치료적 통찰:
역설적이지만, {userName}이 덜 추격하면
{partnerName}이 오히려 다가올 수 있음

### 3. 애착 스타일 분석 (조심스럽게)

**원칙:**
- 확실할 때만 언급
- 항상 "가능성" 표현
- confidence 명시 (low/medium/high)

**증거 기반 추론:**

**안정 애착 (Secure)**
- SUPPORT 태그 균형적
- CONFLICT → RESOLUTION 건강하게
- VULNERABILITY 편안하게
- BOUNDARY 명확하고 존중적

**불안 애착 (Anxious)**
- FEAR 태그 빈번
- PURSUIT 태그 많음
- 확인 요구 (메시지에서 확인)
- 버림받음 두려움

**회피 애착 (Avoidant)**
- AVOIDANCE 태그 패턴
- BOUNDARY 강하게 설정
- VULNERABILITY 어려움
- 거리 두기 선호

**혼란 애착 (Disorganized)**
- FEAR + ANGER 혼재
- 다가갔다 멀어짐 반복
- 일관성 없는 패턴
